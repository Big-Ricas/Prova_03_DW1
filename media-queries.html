Guia de Media Queries

descrição e guia do uso de Media Queries com CSS

O que é um Media Query?

Media queries são filtros que podem ser aplicados em uma estilização com CSS. Elas facilitam a mudança na estilização baseada no tipo ou nas características do dispositivo que está renderizando o conteúdo da página como, por exemplo, a altura, largura, se o dispositivo está sendo usado como touchscreen e etc.

De modo geral Media querries são simples de implementar, tão simples como:

@media (max-width: 600px)
{ .facet_sidebar
background-color: white;
{
display: none;
}
}

(Nesse caso, em resoluções inferiores á 600px o fundo da página será mudado para a cor branca)

No entanto, quando vamos aos fatos, se torna muito trabalhoso ajustar mesmo que seja 1 só resolução a mais, tal como de um monitor 16:9 para um monitor 4:4 (o que não muda muito honestamente, mas a depender da página seria necessário uma alteração para casos assim).

Ao montar um media querry temos muitas possibilidades de condições para as personalizações, Citando um por um vamos por uma breve explicação da função de cada

Operadores Lógicos

Temos Basicamente os operadores lógicos mais comuns, NOT, AND, e ONLY, e com algumas vírgulas podemos ter um resultado semelhante ao OR. O AND é utilizado para combinar medias features com media types e medias features com outras medias features; o NOT é aplicado para toda a media query e simplesmente nega a estilização da lista em que está presente; o ONLY serve para previnir que navegadores mais antigos e que não suportam determinada estilização tentem estruturar algo não possível, utilizado justamente par executar os media querries apenas em navegadores que apresentam as características escolhidas finalmente o OR, que não existe diretamente dentro dos operadores, mas separar as listas por vírgulas têm a sua função, desse modo, quando qualquer uma das querries retornar como verdadeiro, os estilos serão aplicados

Características de mídia

São as "métricas" que serão utilizadas para definir quando que as alterações no estilo serão feitas, sendo assim, as opções são variadas, como se o dispositivo é ou não monocromático, quantos bytes ele apresenta para cara cor, a altura e a largura do aparelho, qual sua resuloção, seu aspecto e sua orientação.

Listando a maioria deles, temos:

  • color
  • color-index
  • aspect-ratio
  • device-aspect-ratio
  • device-height
  • devide-width
  • grid
  • height
  • width
  • monochrome
  • orientation
  • resolution

além desses, alguns navegadores apresentam media querries específicos para utilização quando a navegação é feita por eles, mas isso muda de um para o outro

No caso como exemplo, temos o fundo do site, que tem seu tamanho mudado de acordo com a tamanho da janeja em que é exibido, de modo que ele sempre ocupe todo o espaço livre e não tenha que se repitir para preencher o que falta
